== Q. What order information is imported in OMS?

The following order information is imported in OMS:

* Customer address
* Shipping address
* Billing address
* Company
* Sales channel
* Order Date
* Shopify order ID
* Shopify order name
* Shopify order number
* Shipping method
* Product information
** SKU/Barcode
** Quantity
** Selling price
* Shipping and handling charge for order
* Order total
