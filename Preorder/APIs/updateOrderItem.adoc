= Update Order Item

To update the record of order-item you will need to call `/updateOrderItem` endpoint with post method. Further it updates the record in OrderItem entity using entity-auto.

*Endpoint*
----
POST /api/updateOrderItem HTTP/1.1

Host: example.com
